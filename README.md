# AI-Based Smart Complaint Management System

Monorepo containing the frontend and backend for the Smart Complaint Management System.

## Project Structure

```
├── backend/    # Node.js + Express + MongoDB API
└── frontend/   # React frontend (add your app here)
```

## Backend

See [backend/README.md](backend/README.md) for setup and API documentation.

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

## Frontend

Add your frontend application in the `frontend/` directory.
